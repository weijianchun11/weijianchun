# 智能地址识别系统

基于百度AI开放平台开发的智能地址识别工具，可以自动从文本中提取地址信息，并解析出省、市、区、详细地址、经纬度等结构化数据。

## 📋 项目信息


- **项目类型**：AI应用开发
- **技术平台**：百度AI开放平台 + Python

## 🎯 项目背景

在日常工作和生活中，我们经常需要从大量文本中提取地址信息，如：
- 电商平台的收货地址
- 物流系统的运单信息
- 用户留言中的位置信息
- 新闻资讯中的地点描述

传统的人工提取方式效率低、易出错。本项目利用百度AI的自然语言处理能力，实现了自动化、智能化的地址识别。

## ✨ 主要功能

1. **地址识别**：自动识别文本中的地址信息
2. **结构化解析**：提取省、市、区、详细地址
3. **坐标定位**：返回地址的经纬度坐标
4. **批量处理**：支持多条地址批量识别
5. **交互模式**：支持实时输入和识别

## 🛠️ 技术架构

- **开发语言**：Python 3.7+
- **AI平台**：百度AI开放平台
- **核心SDK**：baidu-aip
- **接口类型**：自然语言处理 - 地址识别API
